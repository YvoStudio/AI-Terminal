# AI-Terminal 功能清单

> 文档更新日期：2026-03-21
> 版本：v0.1.0

## 一、核心终端功能

### 1.1 多标签终端
- [x] 多标签页支持，每个标签独立 PTY 进程
- [x] 标签拖拽排序
- [x] 标签右键菜单（切换 Shell、关闭等）
- [x] 双击标签名重命名
- [x] 标签颜色自定义（右键菜单）

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+T` | 新建标签 |
| `Ctrl+W` | 关闭当前标签 |
| `Ctrl+[` / `Ctrl+]` | 切换左右标签 |
| `Ctrl+1-9` | 跳转到指定位置标签 |

### 1.2 终端渲染
- [x] xterm.js 终端引擎
- [x] WebGL 加速渲染（`@xterm/addon-webgl`）
- [x] Unicode 11 支持（emoji/CJK 字符宽度正确）
- [x] 10000 行滚动缓冲区
- [x] UTF-8 安全字符串处理（防止多字节字符截断）
- [x] 滚动到底部按钮（用户上滚后自动显示）
- [x] 终端内搜索（`Ctrl+F` 触发搜索面板）

### 1.3 剪贴板支持
- [x] **Ctrl+V 粘贴文本**
- [x] **右键点击粘贴文本** ⚠️ *仅 Windows/Linux，macOS 原生右键菜单*
- [x] **Ctrl+V 粘贴图片**（自动保存为文件，路径插入终端）
- [x] **拖拽图片到终端**（自动保存为文件，路径插入终端）
- [x] **拖拽文件到终端**（插入文件路径）

> **平台差异说明：**
> - **macOS**: 右键点击显示系统原生上下文菜单，粘贴请使用 `Cmd+V`
> - **Windows/Linux**: 右键点击直接粘贴剪贴板内容

## 二、AI 感知功能

### 2.1 AI 工具自动检测
- [x] Claude Code 自动检测
- [x] Aider 自动检测
- [x] Cursor 自动检测
- [x] Copilot CLI 自动检测

### 2.2 标签状态指示
- [x] **绿色状态** - 执行中（AI 正在运行命令）
- [x] **黄色闪烁** - 等待输入（AI 等待用户响应）
- [x] **红色圆点** - 完成未读（后台任务完成，需切换查看）

### 2.3 标签自动重命名
- [x] 根据用户输入内容自动重命名标签
- [x] 检测到 5 条用户输入后开始重命名

### 2.4 通知系统
- [x] **macOS**: Dock 图标跳动 + 红色角标数字（待处理标签数）
- [x] **Windows**: 任务栏闪烁通知
- [x] 切换到标签时自动清除角标/通知
- [x] 触发条件：Claude 任务完成（done-unseen 状态）

> **平台差异说明：**
> | 功能 | macOS | Windows | Linux |
> |------|-------|---------|-------|
> | Dock/任务栏角标 | ✅ 红色数字徽章 | ❌ 不支持 | ❌ 不支持 |
> | 图标跳动 | ✅ Dock 图标弹跳 | ⚠️ 任务栏闪烁 | ❌ 不支持 |
> | 通知触发 | 同左 | 同左 | 同左 |

## 十三、构建与开发

### 13.1 开发环境
```bash
npm run tauri:dev    # 完整开发模式（前端 + 后端）
npm run dev          # 仅 Vite 前端开发服务器
cd src-tauri && cargo check  # Rust 类型检查
```

### 13.2 生产构建
```bash
npm run build        # 前端构建
npx @tauri-apps/cli build  # 完整打包
```

### 13.3 输出产物
| 格式 | 路径 | 说明 |
|------|------|------|
| MSI | `src-tauri/target/release/bundle/msi/` | Windows 安装程序 |
| NSIS | `src-tauri/target/release/bundle/nsis/` | Windows 安装程序（备用） |
| DMG | `src-tauri/target/release/bundle/dmg/` | macOS 安装程序 |
| AppImage | `src-tauri/target/release/bundle/appimage/` | Linux 可执行文件 |

### 13.4 图标生成
```bash
powershell -File scripts/gen-icon.ps1
```
- 输入：`scripts/icon-source.png`（1024x1024）
- 输出：`src-tauri/icons/icon.ico`（多分辨率 PNG-in-ICO）
- 尺寸：256 / 128 / 64 / 48 / 32 / 16

## 十四、已知限制

### 14.1 macOS 限制
- 右键粘贴不可用（系统原生菜单），请使用 `Cmd+V`
- Dock 角标仅支持数字，不支持自定义图标

### 14.2 Windows 限制
- 任务栏无角标数字，仅支持闪烁通知
- PowerShell PSReadLine 预测功能会干扰 PTY 渲染（已自动禁用）

### 14.3 Linux 限制
- 无 Dock/任务栏通知支持
- 部分发行版需手动安装 WebKit2GTK
