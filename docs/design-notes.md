# AI-Terminal 设计笔记

## 项目现状（2026-03-12）

基于 **Tauri 2**（Rust 后端 + TypeScript/xterm.js 前端）的跨平台多标签终端，为 AI CLI 工具（特别是 Claude Code）深度优化。

### 已实现功能

#### 核心终端
- 多标签终端，每个标签独立 PTY 进程
- Tab 拖拽排序
- 终端内搜索（⌘F / Ctrl+F，基于 xterm.js SearchAddon）
- 滚动到底部按钮（终端内容不在最底行时自动出现）
- WebGL 加速渲染 + Unicode 11 支持（emoji/CJK 宽度正确计算）
- 剪贴板图片粘贴（自动保存为文件，路径插入终端）
- 10000 行滚动缓冲区
- UTF-8 安全的字符串处理（防止多字节字符截断 panic）

#### AI 感知
- Claude/Aider/Cursor/Copilot 自动检测
- 标签状态指示灯：执行中（绿）、等待输入（黄闪）、完成未读（红）
- 标签自动重命名（根据用户输入内容）
- Claude 检测时自动记录到历史

#### 通知系统
- macOS：Dock 图标跳动 + 角标数字（待处理标签数）
- 切换到标签时自动清除角标
- 触发条件：Claude 任务完成（done-unseen 状态）

#### Claude --resume 集成
- 统一历史面板：Claude 会话 + 工作区历史，按时间排序
- Claude 会话带 `resume` 标签，点击自动新建标签执行 `claude --resume <sessionId>`
- 历史条目鼠标悬停显示删除按钮
- 支持删除 Claude 会话文件和历史条目

#### Notepad（笔记板）
- 按标签独立存储文本块
- 手动发送 / 删除文本块
- **自动提交**：Claude 等待输入时自动发送第一个文本块（300ms 延迟确保 prompt 就绪）
- 可调宽度（拖拽左边框）

#### 命令面板（⌘⇧P）
- 模糊搜索，键盘导航
- 操作：新建/关闭标签、终端搜索、切换笔记面板、清空终端、切换主题、跳转标签

#### 底栏工具
- 选择图片 / 文件 / 目录（路径插入光标处）
- 历史会话面板
- 使用技巧面板
- 主题切换器（10 种主题：Pure Black, One Dark, Dracula, Tokyo Night 等）

#### 平台适配
- macOS：原生交通灯按钮（titleBarStyle: Overlay），隐藏自定义窗口控件
- Windows/Linux：自定义最小化/最大化/关闭按钮，setDecorations(false)
- Windows：默认使用黑体 + 较大字号
- macOS：Menlo 优先 + Nerd Font fallback
- Tab 右键菜单支持 Shell 切换（cmd/powershell/wsl）

#### 数据持久化
- 标签状态保存/恢复（saved-tabs.json）
- 会话历史保存（tab-history.json，最近 15 条）
- 主题选择持久化（localStorage）

---

## AI 支持策略

**Claude 优先，其他 AI 兼容为辅：**

- 核心功能（resume 集成、idle 检测、自动提交）专为 Claude 深度设计
- 其他 AI（Gemini CLI、Aider、Cursor 等）检测到命令后正常开标签、终端正常使用，不强行适配深度功能
- 后续按需评估是否对某个 AI 做深度集成
